软件使用教程

盘点模式

  1. 在首页点击「盘点模式」。
  2. 步骤 1：上传商品清单
    • 点击「上传商品清单」并选择包含 Code / Name / Count 的文件。
  3. 步骤 2：设定日期范围
    • 选择开始 / 结束日期后点「确认」。可重复添加多个不重叠的范围。
    • 日期旁的「×」可删除对应范围。
  4. 步骤 3：点击「完成配置，开始盘点」，系统生成带日期列的统计表。
  5. 盘点过程中可选择两种方式更新数量：
    • 扫描条码：
      1. 点击底部「扫描」> 对准条码（可能需要调整角度）。
      2. 出现「确认条码」窗口，可修改后点击「确认并识别日期」。
      3. 摄像头会切换到日期识别，框住日期直至识别（可能需要调整角度）。
      4. 确认日期后，系统将对应单元格数量 +1。
    • 直接编辑：点击任意单元格，输入数量后失焦即可保存。
  6. 盘点结束后，同样可使用底部「复制」或「导出」输出结果。强烈建议导出。

点货模式

Mode 1 视频即将上线
  1. 在首页点击「点货模式」。
  2. 点击底部「扫描」按钮，首次会弹出摄像头画面。
  3. 对准商品条码，系统识别后会弹出「数量调整」窗口：
    • 如需修改条码，可直接点选并编辑。
    • 使用「+／-」按钮调节数量，或直接在表格里修改。
    • 确认无误后点击「确定」完成添加。
  4. 如条码无法扫描，可点击「手动添加」，在新增行中手动输入 Code 与数量。
  5. 全部录入完成后：
    • 点击「复制」将整表内容复制到剪贴板。
    • 点击「导出」生成 Excel 文件。
