盘点模式

  1. 在首页点击「盘点模式」。Open "Inventory Mode" from home.
  2. 步骤 1：上传商品清单Step 1: Upload product list
    • 点击「上传商品清单」并选择包含 Code / Name / Count 的文件。Choose a file containing Code/Name/Count.
  3. 步骤 2：设定日期范围Step 2: Set date ranges
    • 选择开始 / 结束日期后点「确认」。可重复添加多个不重叠的范围。Select start/end dates and confirm. Multiple non-overlapping ranges are allowed.
    • 日期旁的「×」可删除对应范围。Use "×" to delete a range.
  4. 步骤 3：点击「完成配置，开始盘点」，系统生成带日期列的统计表。Step 3: Click "Finish" to create the table with date columns.
  5. 盘点过程中可选择两种方式更新数量：During inventory there are two ways to update counts:
    • 扫描条码Scan barcode：
      1. 点击底部「扫描」> 对准条码（可能需要调整角度）。Tap "Scan" and aim at the code.
      2. 出现「确认条码」窗口，可修改后点击「确认并识别日期」。Confirm the barcode then scan the date.
      3. 摄像头会切换到日期识别，框住日期直至识别（可能需要调整角度）。Camera switches to OCR; align date until recognized.
      4. 确认日期后，系统将对应单元格数量 +1。After confirming, quantity increases by 1.
    • 直接编辑Edit directly：点击任意单元格，输入数量后失焦即可保存。Click a cell and edit the number.
  6. 盘点结束后，同样可使用底部「复制」或「导出」输出结果。强烈建议导出。After finishing, use "Copy" or "Export" to save results (export recommended).

点货模式Counting Mode

Mode 1 视频即将上线
  1. 在首页点击「点货模式」。Select "Counting Mode" on home page.
  2. 点击底部「扫描」按钮，首次会弹出摄像头画面。Tap "Scan" to open the camera.
  3. 对准商品条码，系统识别后会弹出「数量调整」窗口：Aim at the barcode and adjust quantity:
    • 如需修改条码，可直接点选并编辑。Barcode can be edited manually.
    • 使用「+／-」按钮调节数量，或直接在表格里修改。Use +/- buttons or edit the table.
    • 确认无误后点击「确定」完成添加。Press "Confirm" to add.
  4. 如条码无法扫描，可点击「手动添加」，在新增行中手动输入 Code 与数量。If scanning fails, use "Add Manually" to input code and quantity.
  5. 全部录入完成后：When finished:
    • 点击「复制」将整表内容复制到剪贴板。Click "Copy" to copy the table.
    • 点击「导出」生成 Excel 文件。Click "Export" for an Excel file.
